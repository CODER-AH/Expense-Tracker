# Web Application

This folder contains all the frontend/UI code for the Coorg Trip Expense Tracker.

## Files

- **index.html** - Main HTML structure
- **styles.css** - All CSS styles (1107 lines)
- **app.js** - All JavaScript logic (1731 lines)

## Running Locally

Simply open `index.html` in your browser to test the app locally.

## Deployment

This folder is deployed to GitHub Pages. Any changes pushed to the `main` branch will automatically update the live site.

### GitHub Pages URL
`https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/webapp/`

Make sure to update your GitHub Pages settings to point to the `/webapp` folder.

## Configuration

Update the Google Apps Script URL in `app.js`:
```javascript
const SCRIPT_URL = 'YOUR_APPS_SCRIPT_URL_HERE';
```
